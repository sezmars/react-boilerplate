# React + TypeScript + Vite

## Development server

Run `npm run dev` for a dev server. Navigate to `http://localhost:5173/`. The application will automatically reload if you change any of the source files.

## Techologies

- HTML
- CSS
- TypeScript
- React.js
- Supabase
- React Query
- React Icons
- React router
- Styled components
- Recharts
- Vite
